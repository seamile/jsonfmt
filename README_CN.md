**jsonfmt** 是一款强大的 JSON 处理工具。

🎨 最基本的，你可以使用它来格式化 JSON 数据，并且漂亮的打印出来。

🔄 同时，它也可以将 JSON、XML、YAML、TOML 数据进行互相转化。

🔎 此外，它还可以通过 JMESPATH 或 JSONPATH 来提取 JSON 中的内容。

👀 您甚至可以通过 **jsonfmt** 对两个 JSON 或其他格式的数据进行差异对比。

⚙️ 此外还有其他一些小功能
